"""# Autonomous Swarm Intelligence Lab

**Making swarms intelligent and autonomous.**

Welcome to the GitHub organization of the **Autonomous Swarm Intelligence Lab (ASIL)**.  
We develop open-source research code for autonomous multi-agent systems, swarm intelligence, distributed navigation, cooperative control, learning-based robotics, and safe multi-robot autonomy.

## Research themes

Our repositories support research on topics such as:

- **Swarm intelligence** for autonomous robotic systems
- **Multi-agent navigation** in uncertain and dynamic environments
- **Distributed control and coordination** for robot teams
- **Learning-based planning and control**
- **Safe autonomy** using optimization, Gaussian processes, and model predictive control
- **Formation control and multi-agent stability**

| Repository | Description |
| --- | --- |
